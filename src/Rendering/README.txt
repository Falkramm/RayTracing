Данный пакет содержит код, необходимый для прямого рендеренга кадра и/или последовательности кадров
Класс Camera
Введем упрощение такого вида,
1)  Направление главного луча камеры будет строго параллельно плоскости XY
    в противном случае придется проводить множество других вычислений
2)  Зрение человека ограничено по вертикали от -70(-75) до 55(60)
    для упрощения будем считать, что взгляд вверх и вниз ограничены одинакого
3)  Направление главного луча камеры зафиксированно по координатам (1,0,0)
